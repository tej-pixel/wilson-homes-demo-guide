# Wilson Homes — Sales Cloud Demo Guide

Internal Salesforce AE playbook for walking the Wilson Homes team through their home-builder Sales Cloud POC.

**Live:** https://tej-pixel.github.io/wilson-homes-demo-guide/

## What's in here

- `index.html` — 5-step demo guide with screenshots, click paths, talking points, the Wilson Homes Sales Path reference, an "what's live" table, a quick-reference, and an objection-handling appendix
- `screenshots/` — full captures from the live trial org against seeded demo data

## Build context

- Home builder, Sales Cloud POC: **Lead** = web inquiry, **Account** = community, **Contact** = buyer, **Opportunity** = home/lot deal
- Sales Path (Opportunity, Home Purchase record type): **Registered Interest → Qualified Lead → Tour Scheduled → Negotiating → Closed Won**
- Communities (Accounts): Copper Creek (Fresno), Loma Vista (Clovis), Riverstone (Madera), Granite Park (Clovis), Terra Bella (Fresno)
- 8 reports + the **WH Sales Cloud Overview** dashboard (Total Pipeline, Pipeline by Stage, Community Funnel, Top Reps, Opportunity Age, Stale Deals, Community Pipeline Trend)
- Seeded demo data: 5 communities · 20 home deals across all stages (≈$12M total, $8.4M open) · buyer contacts · web-inquiry leads tagged by buyer type & community · logged activity
- Replaces fragmented Lasso CRM + Mailchimp tooling with one connected platform
